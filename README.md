
# C4K Basketball Data Visualization

An educational helper app created to be used in tandem with the basketball game created by [UVA School of Data Science (SDS)](https://datascience.virginia.edu/news/c4k-hosts-starr-hill-pathway-students-data-analytics-game) in collaboration with [Computers4Kids (C4K)](https://c4kclubhouse.org/).

![](https://api.dsi.virginia.edu/sites/default/files/styles/scale_lg/public/2024-03/C4K.jpg)


## Features

- Available on all platforms (best configured for larger screens such as iPads, ChromeBooks, and laptops)
- Easy-to-use teacher dashboard with ability for students to join via a generated session code.
- Rapidly entered student inputs are updated on the teacher's heatmap in real time.
- Option for teacher to view class-wide or student-specific heatmaps.
